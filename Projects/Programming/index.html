  • Backpropagation

    Research

    During my freshman year at Indiana University I researched neural networks through Undergraduate Research Opportunities in Computing. I setup a neural network in Python and implemented a backpropagation algorithm. From there I tested on xor and other logic tables, and optimized the running parameters to become familiar with the algorithm. During this process I learned to utilize Python's strength in applying mathematical functions. To convey my research I produced a poster presentation.

    Neural networks are one fascinating model for simulating human thought patterns in artificial intelligences.

  • Baseball Reporter

    Application

    Baseball Reporter was my highschool Java class's final project. It generates a baseball game report based on CSV and JSON data formats using statistics about a given game. We worked with Git Hub Organizations, Travis Continuous Integration, and Java for file reading and parsing. As a head designer, my job was to provide data retrieval methods for each data team in the class to procedurally generate the news article's content. In particular, I setup the JSON data retrieval methods and constructed the examples module which I presented to the class.
