Locomotions




It includes multiple locomotions as seen in different VR games so far. It's signature feature is the boosting locomotion, derivative of my Steam game One Giant Leap, and there are other locomotions as well (Skyrim VR -style movement, Tribes Ascend -style skiing, gravity zones, the Interaction System's teleportation, and more). All locomotions are made to play nice with eachother; for example, footsteps cannot be heard while skiing is active, and walking is not possible while using booster jets. I've built the locomotions for quick drag-and-drop setup and checkbox customizability. Template audio is included for each locomotion, from footsteps foley to skiing audio. A lot of modularity is included: cycling between sets of locomotions, multiple input styles, toggling for different functionality such as inertia dampening or terrain feedback, and all sorts of other toggleable modules to provide a customized locomotion setup for as many VR game concepts as possible. Forces and sensitivities can be mapped to different curves. The player has a dynamically shaped body collider. Midair jumping and fuel control can be resource-driven via powerups. I've even added an input module that controls changing between different controller aesthetics.

Boosting

Booster jet propulsion ala One Giant Leap allowing players to fly around like Iron Man in VR; customizable modules for direction, rotation, fuel resources, particle effects, lights, laser-guided aiming, inertia dampening, fall dampening, force diminishing based on raycast distance, antidiminishing (reversed, pulling force toward objects) and more (RAVE MODE)

Skiing

Frictionless movement

Treading

"Smooth movement" best controlled via trackpad input, with walking and running control, and allowing customized speeds, sensitivity, head/wand relativity, footsteps audio, and held/toggled input

Teleportation

Teleportation provided by the SteamVR Interaction System; customize the input using Propulsion Locomotion controls and cycle to enable/disable alongside other locomotion combinations

Jumping

Force-applied jumping, midair-jumping physics, mid-air jumping as a resource with powerups

Turning

Snap-turning with settings for degrees, relativity transform, direction input, and controls input
