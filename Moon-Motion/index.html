Git Started


MoonMotion on GitHub

The MoonMotion repo contains a project template for starting a new Moon Motion project. The Moon Motion Toolkit is included under Assets\Plugins\Moon Motion Toolkit.

I suggest first cloning the repo via Git Hub, then duplicating the repository to create your new project (manually removing any potentially hidden Git files that carry over). Then you can pull updates to the official repository locally as they are released, and copy the changes you want to your project. I would recommend starting a Git Hub repository for your project as well.
